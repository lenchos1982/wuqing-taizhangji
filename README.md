# 无情台账机

> 专门做台账的AI工具：邮件台账、发票台账、合同台账...通通自动化！

## 背景

中小企业日常工作中存在大量重复性台账录入工作：
- 邮件台账：收件人、发件人、主题、日期、附件
- 发票台账：发票号、金额、日期、供应商
- 合同台账：合同号、签约方、金额、期限

传统方式：人工手动录入 → 效率低、易出错

无情台账机：扫描/导入文档 → AI自动提取字段 → 生成结构化台账

## 核心功能

- [x] 文档信息提取（PDF/Word/图片）
- [x] 同义词字段映射（用户手动补充，AI辅助识别）
- [x] 重复字段智能过滤
- [x] 结构化输出（Excel导出）
- [x] 置信度校验 + 人工确认

## 设计思路（核心亮点）

### 1. 用户共建同义词库

- 用户创建台账时定义基础字段（如"公司名"）
- 遇到新变体（如"公司名称"、"企业名称"）时，直接加到该字段的同义词列表
- 工具自动归类，下次同类文件直接识别
- **优势**：用户边用边完善，不用一次性想全；众多个用户累积下来，同义词库越来越全；零标注成本，纯用户驱动

### 2. 置信度校验 + 人工确认机制

- 批处理模式下，置信度低于阈值的字段不阻断，先自动填入
- 处理完成后，生成"需确认"列表
- 用户点击列表项 → 弹出原始文件预览 + 填入内容 → 人工核对/修改
- **优势**：平衡效率与准确率，不打断批处理流程

### 3. 模板市场（多语言）

- 预设常用台账模板（增值税发票、劳动合同、采购合同等）
- 支持简中、繁中、英文三种语言
- 用户可直接选用，降低冷启动门槛

### 4. 数据保密

- 独立存储，不接入飞书或其他第三方SaaS
- 用户数据自主可控
- 企业版可私有化部署（规划中）

## 工作流程

```
用户：创建台账项目 → 定义基础字段
          ↓
导入文档（批处理）→ AI提取字段
          ↓
置信度校验 → 自动填入 → 低于阈值列入待确认列表
          ↓
用户：批量审核待确认项 → 确认/修改
          ↓
生成结构化台账 → Excel导出
```

## 技术栈

- 文档提取：阿里百炼文档智能 / 开源OCR（PaddleOCR）
- LLM：阿里百炼（Qwen系列）
- 存储：独立数据库（SQLite/PostgreSQL）
- 部署：Vercel / 私有化
