# 无情台账机

> 专门做台账的AI工具：邮件台账、发票台账、合同台账...通通自动化！

## 背景

中小企业日常工作中存在大量重复性台账录入工作：
- 邮件台账：收件人、发件人、主题、日期、附件
- 发票台账：发票号、金额、日期、供应商
- 合同台账：合同号、签约方、金额、期限

传统方式：人工手动录入 → 效率低、易出错

无情台账机：扫描/导入文档 → AI自动提取字段 → 生成结构化台账

## 核心功能

- [x] 文档信息提取（PDF/Word/图片）
- [x] 同义词字段映射（自动识别"公司名"="企业名称"="单位名称"）
- [x] 重复字段智能过滤
- [x] 结构化输出（Excel/飞书表格）

## 技术栈

- 文档提取：阿里百炼文档智能/开源模型
- LLM：阿里百炼（Qwen系列）
- 存储：飞书多维表格 / Airtable
- 部署：Vercel
