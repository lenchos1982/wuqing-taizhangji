# 无情台账机

> 专门做台账的AI工具：邮件台账、发票台账、合同台账...通通自动化！

## 背景

中小企业日常工作中存在大量重复性台账录入工作：
- 邮件台账：收件人、发件人、主题、日期、附件
- 发票台账：发票号、金额、日期、供应商
- 合同台账：合同号、签约方、金额、期限

传统方式：人工手动录入 → 效率低、易出错

无情台账机：扫描/导入文档 → AI自动提取字段 → 生成结构化台账

## 核心功能

- [x] 文档信息提取（PDF/Word/图片）
- [x] 同义词字段映射（自动识别"公司名"="企业名称"="单位名称"）
- [x] 重复字段智能过滤
- [x] 结构化输出（Excel/飞书表格）

## 设计思路（核心亮点）

### 1. 用户共建同义词库

- 用户创建台账时定义基础字段（如"公司名"）
- 遇到新变体（如"公司名称"、"company"）时，直接加到该字段的同义词列表
- 工具自动归类，下次同类文件直接识别
- **优势**：用户边用边完善，不用一次性想全；众多个用户累积下来，同义词库越来越全；零标注成本，纯用户驱动

### 2. 工作流程

```
用户：创建台账项目 → 定义基础字段
          ↓
遇到新字段变体（如"公司名称"）→ 加到同义词列表
          ↓
工具：自动识别并归类 → 生成结构化台账
```

### 3. 解决的问题

- 传统方案：一次性标注所有字段 → 费时费力
- 无情台账机：用户在使用过程中自然完善知识库 → 越用越智能

## 技术栈

- LLM：阿里百炼（Qwen系列）
